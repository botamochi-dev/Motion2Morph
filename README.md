# Motion2Morph

MMDボーンの動きを **表情モーフ** または **ボーン揺れ** のキーフレームに変換する VMD 変換ツールです。

## 機能

| 機能 | 概要 |
|------|------|
| **モーフ変換** | ボーンの速度・変位から表情モーフのキーフレームを自動生成 |
| **ボーン揺れ** | スプリング・ダンパーシミュレーションでボーンの揺れキーフレームを生成 |
| **複数ボーン合成** | 複数のドライバーボーンを重み付きで合成 |
| **プレビュー** | 生成結果をグラフで確認してから出力 |

## 必要環境

- Python 3.8 以上
- tkinter（Python 標準ライブラリ）

追加インストール不要で動作します。

## 起動方法

```bash
python motion2morph.py
```

## 使い方

### 1. ファイルを読み込む

- **VMD 読み込み** — 動きのソースとなる VMD ファイルを選択します。
- **PMX 読み込み**（任意）— PMX を読み込むとボーン名・モーフ名がコンボボックスに自動補完されます。

### 2. モーフ変換タブ

| パラメータ | 説明 |
|-----------|------|
| ドライバーボーン | 強度算出に使用するボーン（複数・重み設定可） |
| 表情モーフ | 出力先のモーフ名 |
| 動きの種類 | 位置・回転のどちらを使うか |
| 算出モード | **速度**（激しさ）/ **変位**（原点からのズレ） |
| 感度 | 出力値の倍率（0.1〜5.0） |
| スムージング | ガウス平滑化のウィンドウ幅（フレーム数） |
| 最大値 | 出力値の上限（通常は 1.0） |
| 慣性 / 余韻 | 値が大きいほどじわっと変化してゆっくり戻る |

### 3. ボーン揺れタブ

| パラメータ | 説明 |
|-----------|------|
| ドライバーボーン | 揺れの駆動力算出に使用するボーン |
| ターゲットボーン | 揺れを書き込む対象ボーン |
| 揺れ軸 | X（前後）/ Y（左右）/ Z（傾き）を選択 |
| 揺れ方向 | **双方向**（振り子）/ 片方向（＋のみ） |
| 振幅 | 揺れの最大角度（ラジアン） |
| バネ強度 | 小さいほどよく揺れる |
| 減衰 | 小さいほど長く揺れる |
| スムージング | 外力の平滑化ウィンドウ幅 |

### 4. プレビュー確認 → エクスポート

1. **▶ プレビュー生成** でグラフを確認します。
2. 問題なければ **VMD エクスポート** で VMD ファイルに出力します。
